# Генератор оценочных заданий по программированию

Проект представляет собой веб-приложение для подготовки учебных и контрольных заданий по программированию. Преподаватель выбирает язык, тип задания, уровень сложности, указывает тему и вставляет рабочий исходный код. После этого приложение отправляет запрос во внешний сервис генерации и показывает готовый материал для преподавателя и студента.

## Возможности

- Поддержка языков Python, C++ и Java.
- Генерация заданий по исходному коду преподавателя.
- Восемь типов заданий: код с пропусками, зашумленный код, восстановление порядка строк, объяснение кода, определение результата выполнения, дополнение функции, поиск лишнего фрагмента и сопоставление кода с описанием.
- Проверка структуры ответа перед выводом результата.
- Отдельное отображение материала для студента и правильного ответа для преподавателя.
- Копирование материала для студента в буфер обмена.

## Запуск

Для работы нужен Node.js версии 18 или выше.

```bash
npm start
```

После запуска приложение доступно по адресу:

```text
http://localhost:3000
```

Порт можно изменить через переменную окружения `PORT`:

```bash
PORT=3001 npm start
```

## Настройка

Создайте файл `.env` рядом с `server.js` и укажите параметры доступа к OpenRouter:

```text
OPENROUTER_API_URL=https://openrouter.ai/api/v1
OPENROUTER_API_KEY=your_openrouter_api_key
PORT=3000
```

Если ключ не указан или сервис недоступен, приложение покажет сообщение об ошибке и не будет формировать задание.

## Проверка

```bash
npm run check
```

Команда проверяет синтаксис серверного и клиентского кода.

## Структура проекта

- `server.js` - HTTP-сервер, обработка API-запроса, обращение к OpenRouter и проверка полученного ответа.
- `index.html` - разметка страницы.
- `script.js` - клиентская логика формы, отправка запроса и вывод результата.
- `styles.css` - оформление интерфейса.
- `examples/` - примеры исходного кода для проверки работы приложения.
- `package.json` - описание Node.js-проекта и команды запуска.
- `.env.example` - пример файла настроек без реального ключа.

## Примеры исходного кода

В каталоге `examples` находятся короткие программы на Python, C++ и Java. Их можно использовать для демонстрации работы приложения при защите проекта.
